Location J.A.L

Pour mon travail final du cours 420-345 de programmation web avancée j'ai décidé de faire un site web de location de voiture. Cela permet à la location de s'assurer que le permis du potentiel client est valide et que la période durant laquelle celui-ci souhaite faire une location est disponible.
